Operators

FetchDocument
This operator fetches documents from the configured Cloudant database and sends them as JSON strings to downstream operators.The id and the revision of the document to retrieve can be specified.
ScanChanges
This operator uses the Cloudant changes API to get all changes made to documents in the database.
StoreDocument
This operator stores JSON strings in the configured Cloudant database.The JSON string must be provided as rstring attribute in the input tuple. The attribute name is configrable.Optionally the id and revision can be specified via input attributes. If present, any id and revision information in the document is overwritten.If the docuement is already present in the database, it is overwritten. Otherwise it is created.
