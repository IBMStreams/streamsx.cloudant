Operator FetchDocument

IBMStreams streamsx.cloudant Toolkit > com.ibm.streamsx.cloudant 0.1.0 > com.ibm.streamsx.cloudant > FetchDocument

This operator fetches documents from the configured Cloudant database and sends them as JSON strings to downstream operators.The id and the revision of the document to retrieve can be specified.

This operator has 1 input port and 1 output port.
